# 50/30/20 Personal Budget Tracker
A Python-based personal budget tracker implementing the **50/30/20 rule**:  
- 50% Needs
- 30% Wants
- 20% Savings
Easily track your expenses, visualize your spending with a pie chart, and export your budget data to CSV for record-keeping.

## Features
- Add expenses with optional descriptions
- View category-wise running balances
- Display budget summary with over/under budget indicators
- Pie chart visualization of spending vs budget
- Export expenses to CSV
